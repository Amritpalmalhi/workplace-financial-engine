# Workplace Financial Calculation Engine

A modular Python financial modeling engine designed for enterprise real estate portfolio analysis and workplace optimization.

## Features

- 10-Year Rent & OPEX Projection
- Escalation Modeling (Fixed %)
- NPV Computation
- Scenario-Based Comparison
- Salary Arbitrage Modeling
- FX Rate Lock
- Distance-Based Attrition Risk Modeling
- Fully Unit Tested (Pytest)
- Deterministic & Reproducible Calculations

## Architecture

- Domain-driven design
- Isolated calculation layer
- No framework dependency
- API-agnostic
- Easily pluggable into FastAPI / Databricks
